About This Project

This is a side project I’m building to showcase my evolving skills visually. The idea is to create a portfolio website that automatically tracks and displays my skill stats using dynamic radar charts — like the pentagon shapes you see in many games.

Here’s how it will work:

  • You’ll be able to upload files related to different skill categories (3D, Art, Animation, Game Dev, Design, and more).
  • The website will calculate averages for each skill category based on the uploaded work.
  • It will then automatically redraw the radar chart to visually reflect my current skill levels.
  • Each category’s stats will update live, showing my growth and progress over time.

For now, this landing page is just a teaser to share my vision and keep track of progress. After my IREX exhibition at university, I will dedicate more time and focus on fully building out this site.

Stay tuned — this portfolio will be a living record of my creative journey, updated constantly as I add new work!
